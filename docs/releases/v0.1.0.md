# ReliableTask v0.1.0

Initial public release of ReliableTask.

## Highlights

- Lightweight Spring Boot starter for reliable asynchronous tasks, retries, and compensation workflows.
- Database-backed task lifecycle with MyBatis-Plus and MySQL.
- Worker scheduling, task claiming, retry handling, timeout recovery, and thread pool isolation.
- Admin APIs and a runnable demo application for local evaluation.

## What's Included

- Core task model, task lifecycle states, retry strategies, idempotency SPI, and task submission APIs.
- MySQL schema and MyBatis-Plus task store for task records, logs, worker heartbeat, audit logs, and batch operations.
- Spring Boot auto-configuration for store, executor, admin APIs, metrics, serializer, idempotency, and worker settings.
- Demo project showing transactional task submission, retry scenarios, duplicate submission handling, object payloads, and cURL scripts.
- Project governance files including Apache-2.0 license, README, CHANGELOG, CONTRIBUTING, SECURITY, issue templates, PR template, and CI workflow.
